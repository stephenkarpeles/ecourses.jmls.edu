A plugin which allows you to download your posts, pages, custom post types, comments authors to .xls or .xlsx format.

== Description ==
Wouldn't it be super-convenient if you could export your blog or website to MS Excel? Now it's possible! Use Export 2 Excel to convert your data into native MS Excel formats and experience true portability.

Features:
>1. Supports both WordPress and WordPress Multisite.
>2. Supports both .xls and .xlsx files.
>3. Convert whatever you want: posts, pages, and custom post types, comments authors.
>4. Simplicity redefined: no need for complex settings or writing code.

Install Export 2 Excel now and see how easy it makes everything!


== Installation ==

1. Upload the `export-2-excel` folder to the `/wp-content/plugins/` directory
2. Activate the Plugin through the 'Plugins' menu in WordPress.
3. For WordPress go to `Tools` -> `Export to Excel` and for WordPress Multisite Administrator go to `Settings` -> `Export to Excel`.
4. Select your criteria and download excel.
